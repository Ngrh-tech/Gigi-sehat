# Gigi-sehat
Gigisehat adalah aplikasi web sederhana yang membantu masyarakat di daerah dengan akses terbatas ke klinik gigi untuk melakukan deteksi dini masalah gigi, terutama gigi berlubang (karies). Aplikasi ini menggunakan kamera HP/komputer atau unggah foto untuk menganalisis kondisi gigi secara sederhana berbasis warna.
